**XML JVF Import**

Plugin slouží k importu a náhledu dat ve formátu "_*.jvf.xml_". U importovaných dat je rozpoznána geometrie a jsou automaticky přidány do mapy. Stylování je nastaveno podle souboru "_style_map.py". Součástí je také logika přiřazení atributů z importovaných souborů.

**Použití**

Aktuální pro verzi JVF 1.4.3.

Plugin byl vyvíjen pro starší verze QGIS a je kompatibilní do verze 3.20.3. Novější verze nemusí být plně podporovány.
